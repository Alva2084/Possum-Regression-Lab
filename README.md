# Possum-Regression-Lab
This project analyzes a dataset of possums using linear regression and logistic regression. The goal is to compare relationships between body measurements, age, sex, and populate. From this comparison, we are able to build predictive models.

## Features
- Programming Language: Python
- Data Loading
- Linear Regression Model
- Correlation Analysis
- Sex Classification
- Population Classification
- Preprocess Pipeline
- Tests source code

## Requirements
- Python 3.10 or higher
- PyCharm or Terminal
- pip
  - Install required Packages:
    - pip install pandas scikit-learn pytest

## How to Run
To run the test:
- In the terminal (in terminal for PyCharm as well) bash: pytest -v test_regression.py

To Run the Code:
- In the terminal:
  - bash python main.py
- In PyCharm
  - Run the main.py
